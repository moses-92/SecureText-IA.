# SecureText-IA 🔐✨

Un logiciel de cryptographie autonome développé en Python. Ce script permet de chiffrer et déchiffrer des messages textuels en utilisant un algorithme de décalage dynamique basé sur la table ASCII.

## 🚀 Fonctionnalités
- **Chiffrement sécurisé** : Transformation instantanée de n'importe quel texte en une suite de symboles illisibles.
- **Clé Secrète Numérique** : Utilisation d'une clé personnalisée faisant office de mot de passe pour le déchiffrement.
- **Restauration Intégrale** : Moteur de décryptage capable de retrouver le message d'origine sans perte de données.
- **Léger et Autonome** : Script sans dépendance externe, exécutable instantanément sur n'importe quel environnement Python.
